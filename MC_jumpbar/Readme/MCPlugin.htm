Media Center 12 C# Plugin Template

by: Paul Sinnema

Welcome to the Media Center 12 C# Plugin template.

What do you need?

  1. .NET Framework v2.0.50727 or higher. 
  2. Media Center 12.0.307 or higher.
  3. InnoSetup v5.1.11 or higher (http://www.innosetup.com)

Installation into MC12.

  • In Visual Studio set the Project Type to 'Release'.

  • Press F6 to Build the Project.
  • The destination for the Release Build for this project is set to "..\Installation\Build Files". That's where the DLL will be written to. This is also the location the Inno Setup Compiler will look for your installation files. If you need any other DLL's or installation files for the final version, this is the place to copy them to.
  • In this Project File the default for opening the ''Installation Script.iss' is InnoSetup. A double-click on the file will automatically start the Inno Setup Compiler.

  • In the Inno Setup Compiler press F9 to compile and run the Script.
  • Start Media Center 12 and voila a (still empty) plugin should be started.
